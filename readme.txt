== Description ==

Encourage upgrading to the lastest in technology and security by displaying updates for major browsers your visitor is using, or promote a browser your of choice.  Updated are extracted from NewestBrowser.com daily.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload files to the `/wp-content/plugins/newestbrowser/`.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Choose where to add the Newest Browser widget in the sidebar.
4. Or use the short code [NewestBrowser] in post or page content.
